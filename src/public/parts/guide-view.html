Guide

How to use BOIAuto

1

Add a Bot

Go to the Bot page from the sidebar and click + Add. Enter an optional name, pick a Type, then click Generate Token. A secret token will appear below with copy + connect instructions.

Dual Allows a maximum of 2 access tokens per bot.
Shared Unlimited access tokens. Best for multi-user setups.

After creating the bot, the button changes to Waiting Later with a spinner. When your bot client connects, it becomes Finished.

2

Connect a Bot Client

Copy the bot token from the Add Bot modal. Set BOT_TOKEN in your bot client env, or store it in bot.config.json. Run bun run bot:start. The bot's status will switch from Connecting to Connected once the client is online.

3

Generate Access Tokens

On the Access page, click + Add to create an access token tied to a specific bot. Pick a Type:

Private Single-use access. Max 1 automate per token.
Shared Unlimited automates.
Business Single-use access + price/day (game currency).

Share the access token with the recipient. They can use it to run automates linked to that bot.

4

Add an Automate

On the Automate page, click + Add. Enter your Bearer token, then select an access token from the dropdown. Each access item shows its usage (0/1 for Private/Business, single number for Shared) and type badge. The access type is inherited by the automate.

5

Toggle Automation Features

Each automate card has 3 toggles:

Skill — auto-upgrade skill levels
Training — auto-train units
Work — auto-perform work tasks

When any feature is ON, the card border glows and 3 small dots light up next to the automate label.

6

Dashboard Overview

The Dashboard page shows real-time stats: total automates, total bots, total access tokens, connected bots, automation status (Skill/Training/Work counts), and total balance + diamond. Below the stats are quick-action cards and a list of the latest 5 bots with their connection status.

Tip

Bot status: Connecting (spinner, just added) → Connected (green pulse, client online) → Disconnected (red, client offline).
