Guide

How to use BOIAuto

1

Add a Bot

Go to the Bots page from the sidebar and click + Add. Enter the bot token, optionally give it a name (alias), and select a Type:

Dual Standard bot. Max 2 accounts per bot.
Shared Shared bot. No account limit.
Business Max 2 accounts + requires a $ per day rate.
2

Add an Account

On the Account page, click + Add. Enter your Bearer token, then select one of your registered bots from the dropdown. The Type is automatically inherited from the selected bot — it can only be changed on the Bot page. Account info (name, balance, diamond) is fetched automatically.

3

Toggle Automation Features

Each account card has 3 toggles:

Skill — auto-upgrade skill levels
Training — auto-train units
Work — auto-perform work tasks

When any feature is ON, the card border glows green or blue.

4

Manage Bots

On the Bot list, each row shows: Name, Bot ID, Status (green dot blinking = online, red dot = offline), Token (masked, click eye to reveal), and Actions (Edit + Delete). Click the pencil icon to edit the bot's Name and Type (and Rate per Day for Business).

5

View Account Details

Click any account card to open its detail modal. There you can edit the Bearer token, change the linked Bot, configure Skill/Pay, and view real-time stats (Balance, Diamond, Bot ID, Bot Status). If the bot is offline, the status dot will turn red.

Tip

All tokens are stored locally in your browser only. Nothing is sent to any third-party server.
